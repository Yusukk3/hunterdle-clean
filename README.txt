# HunterDle - pacote de silhuetas

1. Copie a pasta public deste pacote para dentro da pasta do seu projeto HunterDle.
2. No arquivo lib/characters.ts, adicione silhouetteImage em cada personagem.
3. No arquivo components/HunterDleGame.tsx, altere o modo silhueta para exibir a imagem.

Imagens fan-made simples, sem arte oficial.
